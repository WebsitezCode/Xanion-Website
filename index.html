Xanior - Dein Minecraft-Netzwerk

Unser Konzept:


Unser Ziel besteht darin, im Laufe der Zeit möglichst viele Minigames für euch bereitszustellen ohne dabei einen enormen Geldaufwand für den Einzelnen zu konstruieren. Wir möchten euch so einen Server zur Verfügung stellen, auf dem ihr euch wohl fühlen könnt und auf dem die kleinen aber feinen Vorteile eines gekauften Ranges leistbar sind. Wir wollen auf lange Sicht einen Server bieten auf dem ihr mit euren Freunden all eure Lieblingsspiele spielen könnt und noch mehr.
Wir bemühen uns sehr darum die Meinung der Community in unsere Entscheidung zu integrieren, schließlich machen wir diesen Server vorallem für euch und wollen natürlich, dass ihr euch darauf wohl fühlt. Außerdem möchten wir an dieser Stelle noch kurz erwähnen, dass der Server stetig an Minispielen gewinnen wird, um euch mehr Auswahl und neuen Spielspaß zu liefern.

Unsere Versprechen:

Dieser Server ist für euch. Wir wollen das ihr Spaß habt, wir wollen das ihr euch wohlfühlt und das dieser Server zu dem wird, was ihr euch wünscht. Deswegen versichern wir euch folgendes:
ein auf die Community eingehendes Serverteam
für den Server eigens programmierte Plugins
kostengünstige Premiumränge
leistungsstarke Server
Interesse an Verbesserungen
abwechslungsreiche Spielmodi
ein motiviertes und an Gerechtigkeit orientiertes Team
